Song Illustration

An assignment from college to illustrate a song in webpage format. I tried to illustrate it as if you were reading a story or comic as you scrolled down the page. I took this more like an art assignment than a website design assignment, but I definitely did test the limits of what I could and couldn't do with code with this project. In a way, this website served as a springboard to really teach me that I could achieve beautiful things with code as well, not just plain boring websites that function.
(Feb. 2020)

This Website

I created this website because I wanted my own website where I could freely design to my designer heart's content, but website builders like Wix/ Weebly/ Squarespace felt too restricting. Plus, what better to use my CS major on? It was definitely restricting in other ways because I can't use PHP/database materials with Github Pages, but I guess that's the price I pay for a free website server with nearly no strings attached, right?
