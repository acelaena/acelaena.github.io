Software Build

Our software build consists of two major portions, a script that runs our sensors, and a script that collects data from the internet. Both are tied together with another script that runs both at regular intervals to collect data, and finally a script that runs daily to upload all the collected data to the database.

  1. Web Scraper
    • The web scraper collects data from Weather.com on the city of choice, and saves it in a CSV. It's written in Java and mainly makes use of the Jsoup library to parse and traverse the requested webpage HTML.
